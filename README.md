# password-generator
パスワードが生成できますｗｗｗｗｗｗｗ

## Usage
[生成機](https://k239i.github.io/password-generator/)にアクセスします

上のテキストボックスにパスワードの長さ(任意の整数)を入れてください

生成するボタンを押すとビックリ！下のテキストボックスにパスワードが生成されています

[![画像](https://raw.githubusercontent.com/k239i/password-generator/main/7FDC6370-D38E-4B0C-83C1-F6A222E3D7CE.jpeg)](https://raw.githubusercontent.com/k239i/password-generator/main/7FDC6370-D38E-4B0C-83C1-F6A222E3D7CE.jpeg)

あまりパスワードの長さが長すぎたり短すぎたりすると遅くなるかもしれません。

推奨: 8~32767
